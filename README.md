📚 图书管理系统

📌 项目介绍

本项目是基于 SpringBoot + MyBatis + MySQL 开发的后端图书管理系统，遵循 MVC 分层架构设计。项目实现了用户管理、图书管理、借阅与归还等核心业务流程，完整覆盖数据库设计、接口开发、业务逻辑实现。

🛠 技术栈

核心语言：Java 8+
后端框架：SpringBoot 2.x
持久层框架：MyBatis
数据库：MySQL 8.0+
构建工具：Maven
版本控制：Git + GitHub
开发环境：IntelliJ IDEA

✨ 核心功能

用户管理模块：实现用户注册、登录、信息查询，支持普通用户与管理员角色区分
图书管理模块：完成图书信息的增加、删除、修改、查询，实时维护图书在馆 / 借出状态
借阅管理模块：支持图书借阅、归还操作，自动记录借阅时间与归还时间
数据持久化：基于 MySQL 存储用户、图书、借阅记录三大核心数据
标准化接口：提供 RESTful 风格接口，结构清晰、易于扩展与对接前端

🚀 快速运行

环境要求
JDK 8 及以上（推荐 JDK 11）
MySQL 8.0+
IntelliJ IDEA（推荐开发工具）
Maven 3.6+

运行步骤

创建 MySQL 数据库，执行 SQL 脚本创建用户表、图书表、借阅表
修改 application.yml 配置文件，填写本地数据库账号、密码
使用 Maven 导入依赖，启动项目主类
启动成功后，通过接口工具测试各项功能
